# JSfuck-and-hieroglyphy-Decoder-and-ENCODER
JSfuck and hieroglyphy Decoder and  ENCODER

基于jsfuck和hieroglyphy，并且改进了jsfuck对中文字的编码方式，缩小编码之后的体积，借鉴于hieroglyphy，把中文字转换成Unicode编码
并且改进了解码的算法
